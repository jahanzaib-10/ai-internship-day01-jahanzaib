# AI Team and Lifecycle Mini-Task

## 1.  AI project stages

*Business problem understanding:** Firstly idendify the actual problem and figure out what the solution of the problem using AI.
**Data collection:** Gathering the relevant data, information and records of the problem.
**Data cleaning and preparation:** Fixing the errors, missing values and inconsistencies of the collected data so the AI(Computer) understand it.
**Exploratory data analysis:** Deeply analysis the data using statistics and graphs to find the hidden patterns and trends.
**Feature engineering:** Extracting useful information and creating new features from the data to help the AI model trains better.
**Model development:** Train the AI model using actual code and teaching the computer.
**Model evaluation:** Checking how accurately the trained model performs and measuring its errors.
**Model deployment:** Launching the model onto a live server or internet so that regular users and applications can use it.
**Monitoring and improvement:** Continuously keeping an eye on the live model's performance and making improvements over time.

## AI Team Roles

**Data Scientist**
**Main responsibility:** Its main job is to study complex data and build models to solve business problems.
**Important technical skills:** Python, R, basic machine learning, and stats.
**Role in the AI project lifecycle:** Works mostly on model development and data analysis.

 **Data Engineer**
**Main responsibility:** To set up data pipelines and make sure data is stored properly.
**Important technical skills:** SQL, Apache Spark, and cloud storage basics.
**Role in the AI project lifecycle:** Helps in data collection and data cleaning steps.

 **Machine Learning Engineer**
**Main responsibility:** Taking the model made by the data scientist and making it ready for production and real users.
**Important technical skills:** Python, Docker, and APIs.
**Role in the AI project lifecycle:** Main role is during model deployment.

 **Data Analyst**
**Main responsibility:** Looking at datasets to make simple reports and dashboards.
**Important technical skills:** SQL, Excel, and PowerBI.
**Role in the AI project lifecycle:** Helps in understanding the problem and exploring the data.

 **MLOps Engineer**
**Main responsibility:** Managing the deployed models and keeping an eye on system performance.
**Important technical skills:** Docker, Git, and basic monitoring tools.
**Role in the AI project lifecycle:** Works in deployment and monitoring phases.
